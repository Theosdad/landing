# Introduction

Welcome to the **Landing** documentation!

## Getting Started

Get started by exploring our documentation.

### What you'll need

- [Node.js](https://nodejs.org/en/download/) version 18.0 or above

## Start your site

Run the development server:

```bash
npm start
```

The site will start at `http://localhost:3000`.

## Generate a new site

You can start editing the page by modifying `docs/intro.md`. The page auto-updates as you edit the file.
